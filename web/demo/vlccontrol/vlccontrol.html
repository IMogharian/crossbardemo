This is a demonstration of how versatile Tavendo WebMQ is for integration with other applications.

  • Use the controller pads to start and pause the playback of a video file in VLC media player, and to skip back and forth in playlists.
  • All interactions on this demo occur in response to messages routed through an instance of Tavendo WebMQ. Information the instance used is displayed in the status line at the bottom of the demo box.
  • To enable remote control of VLC, you only need to select 'View' - 'Add Interface' - 'Web' from the main menu bar. There is no need to change the default VLC Host set on the demo page, unless you want to control an instance of VLC running on a different machine.
  • You can control the local instance of VLC from any browser or browser tab by entering the displayed controller channel in these other browsers or tabs.
