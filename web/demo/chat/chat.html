A simple chat demo using Tavendo WebMQ

  • All interactions on this demo occur in response to messages routed through an instance of Tavendo WebMQ. Information the instance used is displayed in the status line at the bottom of the demo box.
  • All users on the same channel are connected.
  • Nicks are picked randomly from a set of presets, but can be changed.
